해당 repo의 android/app/src/main/java/com/credittalka/CallStateReceiver.kt 코드 기능 및 출력되는 문구를 다음과 같이 변경하고 싶어.

1) 전화번호부에 없는 번호로 전화가 올 경우
>> "저장되지 않은 전화번호입니다. 금융사기에 주의하세요."

2) 전화번호부에 저장한 지 1주일 이내인 번호로 전화가 올 경우
>> "저장된지 1주일 미만인 번호입니다. 금융사기에 주의하세요."

3) 크레딧톡 DB에 사기 정보가 있는 번호로 전화가 올 경우
>> "피해사례가 등록된 전화번호입니다. 크레딧톡을 참고하세요." 라는 문구를 띄우고, 해당 PUSH 클릭 시 전화 온 번호가 검색되어 있는 src/screens/SearchBaseScreen.js 화면을 출력해줘. (어떤 사기를 쳤는지 열람하기 위함)
